etxt interactive examples

This page links to a few interactive examples from etxt, a text rendering library made for Ebitengine, the open source 2D game engine written in Golang by Hajime Hoshi.

Examples have been slightly adapted and compiled to WASM so they can be executed directly on the web:

  • Words: the example on etxt's README.md. Just words flashing by; nothing special about it.
  • Shaking: fun use of a custom draw function that reacts to the user and the context.
  • Colorful: another use a custom draw function to make text progressively change color.
  • BBCode: a chatroom simulation where the formatting is done through bbcode-like markup, implemented on top of twines through type embedding.
  • Elastic Sizer: make text into a spring by leveraging the Sizer interface.
  • Cutout: an overly fancy example of blend modes being used for cutout text.

You can find the original sources at etxt/examples/ebiten/.
